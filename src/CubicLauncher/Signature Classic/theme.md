# Signature Classic

Tema claro oficial del **CubicLauncher Signature Pack**. Diseñado para ofrecer una experiencia limpia, moderna y legible durante el día, con acentos azules que mantienen la identidad visual de CubicLauncher.

## Características

- **Paleta clara y profesional**: fondos en tonos gris-azulados muy suaves.
- **Acento azul signature**: `#2563eb` para botones, estados activos y glows.
- **Alto contraste**: texto oscuro `#0f172a` sobre superficies claras.
- **Fondo sutil**: degradado generado con un toque del color de acento.
- **Formato v2 (TOML)**: aprovecha el sistema de themes recomendado por CubicLauncher.
