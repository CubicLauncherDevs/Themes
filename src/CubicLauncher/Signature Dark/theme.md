# Signature Dark

Tema oscuro oficial del **CubicLauncher Signature Pack**. Ideal para sesiones nocturnas, con una paleta azulada elegante y un acento vibrante que mantiene la legibilidad sin fatigar la vista.

## Características

- **Paleta oscura profunda**: fondos `#0f172a` y tarjetas `#1e293b`.
- **Acento azul brillante**: `#3b82f6` para resaltar acciones principales.
- **Contraste cuidado**: texto claro sobre superficies oscuras con estados bien diferenciados.
- **Fondo sutil**: degradado oscuro con glow azul suave.
- **Formato v2 (TOML)**: sistema de themes recomendado.
