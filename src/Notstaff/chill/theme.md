# Chill

Un tema relajante con tonos fríos azulados y una estética calmada. Los colores están extraídos del arte de fondo para una experiencia visual coherente y envolvente.

## Características

- **Paleta extraída del fondo:** Colores que armonizan perfectamente con la imagen de fondo
- **Acento azul-cian frío:** Detalles en tonos azulados que guían la atención sin ser agresivos
- **Fondos azul-grisáceos:** Tonos fríos en todas las superficies para una experiencia visual coherente
- **Animaciones sutiles:** Transiciones suaves y efectos de glow en elementos interactivos
- **Scrollbar personalizada:** Scrollbar delgada y elegante en tonos azul-cian
- **Modales animados:** Transición de entrada suave para ventanas modales
- **V2 con Inject.css:** Usa el nuevo sistema de inyección CSS para personalización avanzada
